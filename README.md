# AI Interview Training Platform

AI interview coach designed to help software engineering, AI, ML, and backend candidates succeed in AI screening interviews, technical interviews, and behavioral interviews.

## Goal

Build a production-grade, deployed, resume-worthy product before the end of June 2026.

## Tech Stack

- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: FastAPI, Pydantic, SQLAlchemy, Alembic
- Database: PostgreSQL
- AI: Groq API
- Auth: JWT
- Deployment: Vercel, AWS EC2
- Payments: Lemon Squeezy

## Core Features

- User authentication
- Resume-based interview preparation
- Job-description-based interview preparation
- Adaptive AI interview sessions
- Evaluation and coaching
- Session history
- Progress tracking
- Interview reports
