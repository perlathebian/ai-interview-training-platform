# Week 1 Plan

## Goal

Build the foundation of the AI Interview Training Platform.

By the end of Week 1, the app should support:

- local full-stack development
- user authentication
- resume context
- job target context
- adaptive text interview sessions
- AI evaluation and coaching
- session history
- basic reports
- tests
- CI preparation

## Week 1 Scope

In scope:

- React frontend setup
- FastAPI backend setup
- PostgreSQL setup
- JWT authentication
- core database models
- text interview MVP
- basic AI agents
- GitHub workflow

Out of scope:

- voice interviews
- video interviews
- payments
- PDF export
- admin dashboard
- advanced analytics
